Each command is:
FUNCTIONNAME|Arg1|Arg2|Arg3|...|ArgN
followed by \n.

Commands:

* getTitleInformation|MOVIE_TITLE
    Print-out from handbrake showing track/chapter/audio break-down.
getTitles
    List of titles as:
	MOVIE_TITLE|HAS_RIPPED 1/0|HAS_ENCODED 1/0|HAS_UPLOADED 1/0|VIDEO_TRACK|AUDIO_TRACKS (seperated by , or 'auto')
addISO|FILE_NAME
    Adds ISO to queue and names it using standard transformation.
setVideoTrack|MOVIE_TITLE|TRACK_NUMBER
    Sets video track of MOVIE_TITLE to TRACK_NUMBER.
ripStatus
    String of rip status.
encodeStatus
    String of encode status.
uploadStatus
    String up upload status.
